# Simple ML Student Predictor

## Description
This project is a simple machine learning application built with Python to predict student approval based on input data.

## Features
- Data preprocessing
- Model training
- Prediction output

## Technologies
- Python
- Pandas
- Scikit-learn

## How to Run
```bash
git clone https://github.com/rubensortizjr-arch/simple-ml-student-predictor.git
cd simple-ml-student-predictor
pip install -r requirements.txt
python main.py
```
